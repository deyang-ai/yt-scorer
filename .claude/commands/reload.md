# Reload Extension in Chrome

Instructions to reload YT Scorer after a build:

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer mode** (top-right toggle) if not already on
3. Find the **YT Scorer** card
4. Click the circular ↻ **reload** button on the card
5. Navigate to any YouTube video page
6. Click the YT Scorer icon in the Chrome toolbar to open the popup

If the extension isn't loaded yet:
- Click **Load unpacked**
- Select the `dist/` folder inside this project
